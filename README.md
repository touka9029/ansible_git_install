# 使い方
```
# コンテナ起動
$ docker-compose up -d --build

# コンテナに対してansible-playbookを実行
$ docker exec -it ansible_test ansible-playbook developer_setup.yml
```

- ansibleの内容
  - gitをソースコードからインストール
  - バージョンを確認
  - gitをアンインストール
